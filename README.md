# すこやか整形外科医院 公式Webサイト

石川県能登地区に2026年春開院予定の整形外科医院の公式Webサイトです。

## サイトURL

- **ホームページ**: https://sukoyaka-ortho-clinic.com/sukoyaka-homepage-green.html
- **採用ページ**: https://sukoyaka-ortho-clinic.com/

## ページ一覧

| ページ | ファイル | 説明 |
|-------|---------|------|
| ホームページ | sukoyaka-homepage-green.html | 医院の紹介、診療案内、アクセス情報 |
| 採用ページ | index.html | スタッフ募集情報 |
| 当院について | about.html | 医院の詳細情報 |

## 特徴

- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- 緑を基調とした落ち着いたデザイン
- 丸ゴシック体による親しみやすい印象
- モバイル固定バー（電話・WEB予約ボタン）

## クイックスタート

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# デプロイ
npm run deploy
```
