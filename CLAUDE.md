# プロジェクト情報

## Webサイト開発プロジェクト
このプロジェクトは様々なWebサイトとWebアプリケーションを作成するためのベースプロジェクトです。

## 技術スタック
- Vite (開発サーバー・ビルドツール)
- GitHub Pages (ホスティング)
- Claudeのフロントエンドデザインスキル活用

## デザイン原則
- 個性的なタイポグラフィ (Playfair Display + DM Sans)
- カスタムテーマシステム
- モーションデザイン重視
- 視覚的深みのある背景

## コマンド
- `npm run dev` - 開発サーバー起動
- `npm run build` - プロダクションビルド
- `npm run deploy` - GitHub Pagesへデプロイ

## ページタイプ
- ランディングページ（LP）
- コーポレートサイト
- ポートフォリオ
- ブログ
- Eコマース
